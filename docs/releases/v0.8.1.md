# Gancho v0.8.1 — keep Pro unlocked and recover sync predictably

Gancho 0.8.1 is a reliability release. It keeps a lifetime Pro purchase unlocked
through transient StoreKit gaps, makes CloudKit recovery decisions deterministic,
and strengthens the evidence the release lane produces.

## Highlights

- **A lifetime Pro purchase stays unlocked.** StoreKit occasionally omits a valid
  non-consumable purchase from its current-entitlements snapshot. Gancho no
  longer treats that transient gap as a lost entitlement, so Pro features remain
  available instead of briefly relocking.
- **Sync recovery is predictable.** CloudKit recovery decisions and pending-work
  planning moved into focused, deterministic components. Retry and restart
  behavior is unchanged, but the sync engine now carries far less hidden state,
  which makes recovery reproducible and testable.
- **Release validation keeps its evidence.** macOS and iOS UI result bundles are
  preserved, and their screenshots and attachments can be exported as
  reproducible review evidence. Hosted UI validation now verifies final state and
  relative window behavior instead of depending on transient accessibility flags
  or display-specific pixel dimensions.

## Install or update

- Download the signed and notarized
  [Gancho 0.8.1 DMG](https://github.com/johnny4young/gancho/releases/download/v0.8.1/Gancho-0.8.1.dmg),
  or install the Mac app plus its bundled `gancho` CLI with Homebrew:

  ```bash
  brew tap johnny4young/tap
  brew install --cask gancho
  ```

- Existing direct-download installs receive v0.8.1 through the signed Sparkle
  feed. You can also choose **Settings → General → Check for Updates…**.
- The release assets include `Gancho-0.8.1.dmg.sha256` for independent checksum
  verification.

Gancho requires macOS 26 or later. The iPhone and iPad targets require iOS 26
or later and are not distributed by the DMG or Homebrew cask.

## Availability

The direct-download app is signed with a production CloudKit and Push profile,
so the artifact is technically sync-capable. A fresh public install currently
runs as the Free tier and remains local-only because cross-device sync requires
Pro and secure direct-download activation is not yet available. Local encrypted
history persists across app updates.

## Release verification

The published DMG ships with a `Gancho-0.8.1.dmg.sha256` sidecar that pins the
exact bytes; the Homebrew cask is generated from that same checksum, so both
channels resolve to one artifact. Automated release QA verifies the Developer ID
signature, hardened runtime, notarization and stapling, Gatekeeper acceptance,
embedded production profile, signed CloudKit and Push entitlements, checksum
sidecar, and signed Sparkle appcast before anything is published. Manual
clean-install, upgrade, and two-device sync exercises are tracked separately and
are not claimed by this verification result.
